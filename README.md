Structured SQL practice repository designed to bridge foundational and intermediate concepts into real-world business analysis. 
This portfolio demonstrates how SQL is applied to solve practical data problems faced by analysts in domains such as sales, 
customer insights, operations, and reporting.

Coverage includes:
- Core Querying: SELECT, WHERE, ORDER BY, GROUP BY, HAVING, DISTINCT, TOP
- Filtering & Operators: Comparison (=, <>, >, <, >=, <=), Logical (AND, OR, NOT), Range (BETWEEN), Membership (IN, NOT IN), Search (LIKE)
- Joins: INNER, LEFT, RIGHT, FULL, CROSS, Anti Joins, Multi-table joins
- Set Operators: UNION, UNION ALL, INTERSECT, EXCEPT
- Window Functions: Aggregations (SUM, AVG, COUNT, MIN, MAX), Ranking (ROW_NUMBER, RANK, DENSE_RANK, NTILE), Value Functions (LEAD, LAG, FIRST_VALUE, LAST_VALUE)
- String Functions: CONCAT, SUBSTRING, LEFT, RIGHT, REPLACE, LEN, CHARINDEX, PATINDEX, STUFF, FORMAT
- Date Functions: DATEPART, DATENAME, DATEDIFF, EOMONTH, GETDATE, SYSDATETIME
- Conditional Logic: CASE WHEN for categorization, mapping, handling NULLs, conditional aggregation
- Language Components: DQL (querying), DML (INSERT, UPDATE, DELETE, TRUNCATE), DDL (CREATE, ALTER, DROP, COMMENTS)
